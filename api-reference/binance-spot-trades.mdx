# Recent Spot Trades

Returns recent Binance Spot trades for a supported coin.

<Note>
  Requires a **Pro** or **Enterprise** plan.
</Note>

## Request

```
https://api.polyhistorical.com/v1/{coin}/spot/trades
```

### Headers

<ParamField header="X-API-Key" type="string" required>
  Your API key. See [Authentication](/authentication).
</ParamField>

### Path parameters

<ParamField path="coin" type="string" required>
  Coin symbol. Supported values: `btc`, `eth`, `sol`.
</ParamField>

### Query parameters

<ParamField query="limit" type="integer" default="100">
  Number of trades to return (1-1000).
</ParamField>

## Response

<ResponseField name="data" type="array">
  List of trade objects.
</ResponseField>

<ResponseField name="total" type="integer">Number of trades returned.</ResponseField>
<ResponseField name="limit" type="integer">Requested result limit.</ResponseField>

## Example

```bash cURL
curl -H "X-API-Key: YOUR_API_KEY" \
  "https://api.polyhistorical.com/v1/btc/spot/trades?limit=100"
```
